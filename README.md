## 1. 프로젝트 개요

### 1.1 목표

- **데이터 분석의 전체 흐름**을 이해하고 응용할 수 있다.
- 요구사항에 맞는 **데이터 수집·전처리**를 수행할 수 있다.
- **시각화**를 통해 데이터를 직관적으로 표현할 수 있다.
- 전처리 → 분석 → 시각화 → (선택) AI 기반 예측까지의 **실무와 유사한 데이터 처리 경험**을 쌓는다.

### 1.2 데이터 사이언스 프로세스

1. **문제 정의** – 해결하고자 하는 문제 정의  
2. **데이터 수집** – 필요한 데이터 수집 (Kaggle 등)  
3. **데이터 전처리(정제)** – 결측치·이상치 제거, 형식 변환  
4. **데이터 분석** – 전처리된 데이터에서 정보 추출  
5. **결과 해석 및 공유** – 시각화 후 의사결정에 활용

### 4.1 기본 기능 (필수)

| 번호 | 분류 | 요구사항명 | 상세 | 우선순위 |
